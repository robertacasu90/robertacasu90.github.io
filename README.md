Personal portfolio of **Roberta Casu** — Microsoft 365 Solution Developer / Technical Lead (SPFx, React, TypeScript, .NET, Power Platform).

Hand-written HTML, CSS and vanilla JavaScript — no frameworks, no build step. Typography: Fraunces + Inter + IBM Plex Mono. Custom bordeaux/cream/gold palette shared with the project demos linked from the site, so everything reads as one coherent ecosystem.

## Structure

```
index.html        single-page site (experience, projects, skills, education, contacts)
styles.css        all styling, CSS custom properties for the palette
script.js         scroll thread animation and small interactions
assets/           project screenshots and demo GIFs
```

## Run locally

No tooling required — open `index.html` in a browser.
